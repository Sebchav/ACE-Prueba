## Prueba Técnica

Aplicación desarrollada utilizando Python FastAPI para el backend, React para el frontend y MongoDB como base de datos.

## Instrucciones de instalación
Clonar el Repositorio
git clone https://github.com/Sebchav/ACE-Prueba.git
cd ACE-Prueba

## Configuración del Frontend
Dirígete a la carpeta del frontend:

cd frontend

## Instala las dependencias:
npm i

## Corre el frontend:
npm run dev

## Configuración del Backend
Abre la carpeta del backend en Visual Studio Code:

## Crea un entorno virtual (venv):

python3 -m venv venv

## Selecciona el intérprete venv en Visual Studio Code.

## Ejecuta el proyecto del backend con el siguiente comando:

uvicorn main:app --reload

## Con estos pasos, el frontend y el backend de tu aplicación deberían estar listos para su ejecución local.
